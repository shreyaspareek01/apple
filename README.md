# Apple iPhone 15 Pro Website Clone

A high-performance, visually stunning clone of the Apple iPhone 15 Pro website. This project leverages modern web technologies to recreate Apple's iconic design, featuring interactive 3D models, smooth scroll-triggered animations, and a fully responsive interface.

## 🚀 Features

- **Interactive 3D Models**: Detailed 3D rendering of the iPhone 15 Pro using Three.js and React Three Fiber.
- **GSAP Animations**: Complex, high-performance animations powered by GreenSock (GSAP) for a premium feel.
- **Dynamic Content**: Seamless integration of video highlights and interactive features.
- **Responsive Design**: Optimized for all devices using Tailwind CSS.
- **Modern UI/UX**: Clean, minimalist design following Apple's aesthetic standards.

## 🛠️ Tech Stack

- **Framework**: [React.js](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Animations**: [GSAP](https://greensock.com/gsap/)
- **3D Graphics**: [Three.js](https://threejs.org/) & [React Three Fiber](https://docs.pmnd.rs/react-three-fiber)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shreyaspareek01/apple.git
   ```

2. Navigate to the project directory:
   ```bash
   cd appleWebsite
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Project

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.
